# Field Data Collection & Sync Tool with Chatbot

## 📌 Project Overview
This project is a **Single Page Application (SPA)** built using **React, TypeScript, Vite, and Tailwind CSS**.  
It enables **field data collection, record management, and chatbot assistance** for users.  
The frontend is designed to be **responsive, simple, and professional**.

## 🛠️ Technologies Used
- **React.js** (Frontend framework)  
- **Vite** (Build tool for fast development)  
- **TypeScript** (Type-safe coding)  
- **Tailwind CSS** (Modern, responsive UI)  
- **Lucide React** (Icons for chatbot, user, send button)  

---

## 🚀 Features
- SPA navigation (no page reloads)  
- Responsive UI (desktop + mobile support)  
- Integrated chatbot panel  
- Records stored and fetched via REST API
